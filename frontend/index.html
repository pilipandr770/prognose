Прогнозируй события. Собирай виртуальный портфель. Расти в двух рейтингах.

Prediction mode

Отдельный кабинет для событий, прогнозов и модерации user-created рынков.

Пользователь предлагает событие или выбирает уже одобренное.

Каждое предложенное событие проходит модерацию.

Позиции открываются только на игровые евро.

Portfolio mode

Отдельный кабинет для виртуальных сделок и приватного управления портфелем.

Пользователь торгует в игровом окружении без реального брокера.

Публично виден только summary, а не структура позиций.

Portfolio leaderboard не смешивается с prediction leaderboard.

Два разных рейтинга

Рейтинг должен усиливать понятность, а не мешать навигации.

Prediction leaderboard показывает силу в событиях и точности прогнозов.

Portfolio leaderboard показывает доходность и качество виртуального управления.

Из рейтингов пользователь переходит в social layer: подписки, профили и общение.

После регистрации

Пользовательский путь короткий и читаемый.

1. Войти или зарегистрироваться.

2. Открыть личный кабинет и выбрать prediction room или portfolio room.

3. Следить за отдельными рейтингами.

4. Из рейтингов перейти в сообщество и добавить интересных пользователей.
